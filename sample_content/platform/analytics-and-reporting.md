# Fourth Analytics & Reporting

## Overview

Fourth's analytics platform provides real-time workforce intelligence to help operators make data-driven decisions about labor costs, scheduling, compliance, and employee performance. With **50+ standard reports**, custom report building, and AI-powered dashboards powered by Fourth iQ, Fourth gives every stakeholder - from GMs to C-suite - the visibility they need.

## Custom Reporting

Fourth supports custom report building:
- **Report Builder**: Drag-and-drop interface for creating custom reports from any data in the platform
- **Scheduled Delivery**: Email reports on any schedule (daily, weekly, monthly) to any distribution list
- **Export Formats**: PDF, Excel, CSV, and direct API access
- **API Access**: All report data available via Reports API for BI tools (Tableau, Power BI, Looker)

## Data & AI Capabilities (Fourth iQ)

### Demand Forecasting
- Uses 2+ years of historical sales data
- Incorporates weather, local events, holidays, seasonality
- Machine learning models retrained weekly
- 15-minute interval granularity
- **95%+ accuracy** for established locations
- Continuously improves as more data is collected

### Predictive Analytics
- **Turnover prediction**: Fourth iQ identifies at-risk employees based on schedule patterns, tenure, and engagement signals
- **Demand anomaly detection**: Flags unusual patterns for manager review (e.g., unexpected slow periods, event-driven spikes)
- **Optimization recommendations**: AI-suggested schedule changes to improve labor efficiency without impacting service quality
- **Cost forecasting**: Projected labor costs for upcoming schedules before publication

### Cross-Module Analytics
Fourth's unified data model enables analytics that span modules:
- Hiring cost vs. retention rate correlation
- Schedule flexibility impact on turnover
- Training completion impact on productivity
- Fuego (EWA) usage correlation with retention

## Frequently Asked Questions

**Q: Can we build custom reports?**
A: Yes. Fourth includes a drag-and-drop report builder that allows you to create custom reports from any data in the platform. Reports can be scheduled for automatic delivery via email and exported in PDF, Excel, or CSV format.

**Q: Does Fourth integrate with our BI tools?**
A: Yes. Our Reports API provides programmatic access to all report data. Customers commonly integrate with Tableau, Power BI, and Looker. Enterprise customers also have direct database access options for advanced analytics.

**Q: How real-time is the data?**
A: Labor cost and scheduling data updates in real-time as clock events and schedule changes occur. Financial aggregations refresh every 15 minutes. Historical reports are available within 24 hours.

**Q: How does Fourth iQ improve over time?**
A: Fourth iQ's machine learning models retrain weekly using your latest data. Forecast accuracy improves as the system accumulates more historical data, local event information, and seasonal patterns specific to each location.
